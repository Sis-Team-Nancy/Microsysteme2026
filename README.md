# Microsysteme2026
This year, first-year master's students in biomedical engineering collaborated on a common theme: evaluating an athlete's VO2max outside the context of an exam supervised by a sports physician. Divided into 8 projects to collect or display the parameters, the students helped each other both on the technical and physiological aspects, in order to develop functional solutions that will be continued by future students.
